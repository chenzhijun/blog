最近经常看到一些有意思的快捷键，但是又不是经常用到，平常经常用的肯定都能很熟悉了。

idea 方法show outline 类似eclipse `ctrl o`, idea中为`ctrl F12`

idea 有用的插件：`key promoter`，`idea vim`，`lombok plugin`，`maven helper`，`sonar lint`，`alibaba java code guide`
